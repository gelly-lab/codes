# codes

This repository contains a simple JavaScript implementation of Tetris.

## How to run

1. Open `index.html` in a modern web browser.
2. Control the falling blocks with the arrow keys.
   - `ArrowLeft`: move left
   - `ArrowRight`: move right
   - `ArrowDown`: drop faster
   - `q`: rotate counter-clockwise
   - `w`: rotate clockwise

## Development

Install Node.js dependencies (none currently) and run tests:

```bash
npm test
```

Have fun!
